## What is ALFA APY?

ALFA APY is the first yield optimization DeFi protocol built on the Internet Computer Protocol (ICP) utilizing **ckBTC**, similar to Yearn Finance on Ethereum. The protocol automatically maximizes returns for users by finding the highest-yielding liquidity pools and dynamically reallocating assets to optimize yield.

### Key Features

- **ğŸš€ Automated Yield Optimization** - Automatically moves assets between liquidity pools to maximize APY
- **ğŸ’ Multiple Asset Strategies** - Supports ckBTC, ICP, and stablecoin pairs
- **ğŸŒŠ Multi-Provider Integration** - Works with KongSwap, ICPSwap, and other DeFi providers
- **ğŸ”’ Non-custodial** - Users maintain full control of their assets
- **âš¡ Smart Rebalancing** - Dynamic allocation based on real-time APY data
- **ğŸ’° Low Fees** - Minimal costs compared to manual rebalancing

- **Backend**: Rust + Internet Computer Protocol (ICP)
